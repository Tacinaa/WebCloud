Politique IAM appliquée

Le compte de service dispose uniquement de s3:GetObject, s3:PutObject et s3:ListBucket. La permission s3:DeleteObject est volontairement absente — toute tentative de suppression retournera une erreur AccessDenied (403).

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "AllowReadWrite",
      "Effect": "Allow",
      "Action": [
        "s3:GetObject",
        "s3:PutObject",
        "s3:ListBucket"
        // s3:DeleteObject — ABSENT
      ],
      "Resource": [
        "arn:aws:s3:::{{ bucket }}",
        "arn:aws:s3:::{{ bucket }}/*"
      ]
    }
  ]
}
