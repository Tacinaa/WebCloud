Informations Architecture

🔒 Sécurité réseau

  • VPC isolé (10.0.0.0/16)
  • 3 sous-réseaux distincts
  • Security Groups restrictifs
  • Network ACL par couche

🌐 Flux autorisés

  • Internet → Web (80/443)
  • Web → Backend (5000)
  • Backend → PostgreSQL (5432)
  • Pas d'accès direct à la BDD

🐳 Conteneurisation

  • Nginx (reverse proxy)
  • Flask + Gunicorn (WSGI)
  • PostgreSQL 16
  • Docker sur EC2
